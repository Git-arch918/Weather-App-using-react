# Weather-App-using-react

A simple weather app built with React and Material-UI that fetches weather data from OpenWeatherMap API.

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Get an API key from [OpenWeatherMap](https://openweathermap.org/api)
4. Create a `.env` file in the root directory and add your API key:
   ```
   VITE_WEATHER_API_KEY=your_actual_api_key_here
   ```
5. Start the development server: `npm run dev`

## Features

- Search for weather by city name
- Displays temperature, humidity, min/max temp, and weather description
- Responsive design with Material-UI components
- Dynamic weather icons and background images
